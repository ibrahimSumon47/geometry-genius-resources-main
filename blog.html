1. What is DOM and what is the purpose of DOM?

Ans: DOM full name is Document Object Model. DOM represents the document as nodes and object and by doing that programming languages can interact with the page.
DOM is not a programming languages but a web API for websites and without it JS can't be uses in website.

2. How will you select HTML elements using DOM? Name the DOM methods.

Ans: By using HTML elements within a document.
There are total 5 methods in DOM. which is
1. getElementByID().
2. getElementByClassesName().
3. getElementsByName().
4. getElementsByTagsName().
5. querySelector().

3. What is event bubble?

Ans: An event received by an element does not stop one element or move to up parent of the element or move up to ancestor of the element is called event bubble.

4. What is a callback function and why will you use it?

Ans: A callback function is a function pass into another function as an argument.
Callback function needed when many JS action are not synchronous, for event handling, high order function. By using callback function we can easily write our program.
